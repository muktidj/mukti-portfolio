About

Nama Saya Mukti Dwi Jatmoko, saya lahir di Jakarta, 5 Maret 1998. Saya tinggal di Pondok Cabe, Tangerang Selatan, Banten. Di Pondok Cabe pula saya tumbuh dan dibesarkan sampai sekarang. Saya pernah belajar di SDN Cirendeu II,Tangerang Selatan. SMP Darussalam, Jakarta Selatan, SMKN 20, Jakarta Selatan. Saat ini saya melanjutkan pendidikan di Universitas Budi Luhur, jurusan Sistem Infromasi. Saya adalah seorang penggemar teknologi, saya mencoba berbagai teknologi baru. Seperti Linux, Virtual Machine, Git, Docker. Saat ini saya sedang mempelajari HTML, CSS, JavaScript, Python.

Saya fleksibel dalam jam kerja saya, bisa bekerja dimalam hari dan akhir pekan. Saya selalu bersemangat untuk mempelajari keterampilan baru. Saya mampu beradaptasi dengan baik dimanapun saya berada, saya memiliki kemampuan komunikasi yang baik, terkadang ini sulit dimiliki oleh seorang yang menghabiskan waktunya berjam-jam didepan komputer. Meskipun saya menganggap serius pekerjaan saya, saya memiliki humor yang bagus . Saya termotivasi untuk berolahraga sebelum bekerja agar bugar dan sehat. Saat ini saya sedang fokus mencari pekerjaan dibidang Front End Developer dan Web Design.
